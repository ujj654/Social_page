# Steps to Run the Project:-

### 1. Clone the repository:  
git clone \<repository-url\> 

### 2. Set up your .env file:

MONGO_URI = mongodb://localhost:27017/\<YOUR DATABASE NAME\>  
JWT_SECRET = your_secret_key  
PORT = 5000

### 3. Install dependencies:
   npm install

### 4. Start the server:

   node server.js   
        OR  
   nodemon server.js
